# Rediseño y Optimización Web: Estudio Griffa

Este repositorio contiene el código fuente para el nuevo sitio web de **Estudio Griffa** (Adjusters - Marine Surveyors & Consultants), un proyecto de transformación digital enfocado en la modernización de la identidad visual y la optimización del rendimiento para una firma con más de 60 años de trayectoria.

## 📊 Descripción del Proyecto

El objetivo principal fue migrar un sitio web obsoleto (HTML estático de 2014, no responsivo) hacia una arquitectura moderna basada en el **Jamstack**, priorizando la velocidad de carga, la experiencia de usuario (UX) corporativa y la captación eficiente de clientes.

Este proyecto forma parte de mi portfolio profesional, demostrando capacidad para auditar, rediseñar y desplegar soluciones web integrales con un enfoque en la calidad de los "assets" y la eficiencia técnica, habilidades transferibles al manejo y visualización de datos.

## 🛠️ Stack Tecnológico y Metodología

Para este proyecto, se implementó un flujo de trabajo moderno y escalable:

* **Frontend:** HTML5 semántico, **Tailwind CSS** (para un diseño rápido y limpio vía CDN).
* **Gestión de Assets:** Limpieza y escalado de logotipos e imágenes corporativas (`buque.jpg`, `eg-logo-hd.png`) para garantizar calidad HD sin sacrificar rendimiento.
* **Captación de Datos:** Integración con **Formspree** para la gestión de formularios de contacto, recolectando datos estructurados (CSV/JSON) para futuros análisis.
* **Despliegue (CI/CD):** Arquitectura Jamstack desplegada en **Netlify**, con integración continua desde este repositorio de GitHub.
* **Seguridad:** Implementación automática de certificado SSL/TLS.
